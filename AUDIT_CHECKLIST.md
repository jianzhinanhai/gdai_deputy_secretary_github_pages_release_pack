# 上线审计清单

## A. 内容审计

- [ ] 页面标题不含版本号。
- [ ] 页面正文不含“版本、基准、适用、附录、SOP、评分器、项目组合、整改、修订、勾选、checkbox”。
- [ ] 页面不包含给操作人的说明。
- [ ] 页面不包含过程性讨论、修改建议、生成说明。
- [ ] 页面面向副秘书长本人，语气自然、正式、低压力。

## B. 功能审计

- [ ] 顶部目录可以跳转。
- [ ] 搜索框可以输入关键词。
- [ ] 搜索结果可以高亮。
- [ ] “上一个”功能正常。
- [ ] “下一个”功能正常。
- [ ] “清除”功能正常。
- [ ] 展开/收起功能正常。
- [ ] 底部“回到顶部”功能正常。
- [ ] 右下角悬浮回到顶部功能正常。

## C. 手机端审计

- [ ] iPhone 竖屏下第一屏不被标签和搜索框大量遮挡。
- [ ] Android 竖屏下正文可读。
- [ ] 顶部导航不覆盖正文。
- [ ] 表格可读或可横向滚动。
- [ ] 按钮尺寸适合触控。
- [ ] 页面无横向整体溢出。
- [ ] 搜索跳转后命中内容不被顶部栏遮挡。

## D. GitHub Pages 审计

- [ ] `index.html` 位于仓库根目录。
- [ ] `.nojekyll` 位于仓库根目录。
- [ ] GitHub Pages Source 设置为 GitHub Actions。
- [ ] Actions workflow 成功。
- [ ] 线上 URL 可公开访问。
- [ ] HTTPS 正常。
